# InfoCollector

InfoCollector is a simple C++ console application that allows users to input and display details about different categories such as students, credit cards, bikes, and books. This project is designed for learning basic C++ inputoutput operations, variables, and data types.

## Features

- Student Information  
  Collects and displays the name, ID, email, college name, and phone number of a student.

- Credit Card Information  
  Collects and displays the cardholder's name, card number, valid till date, and CVV.

- Bike Information  
  Collects and displays details of a bike including name, model, engine CC, cost, on-road cost, and mileage.

- Book Information  
  Collects and displays details of a book including book name, cost, publication date, author name, publisher name, and number of pages.

## Prerequisites

- C++ compiler (e.g., GCC, MinGW, or Visual Studio)
- Basic knowledge of running console programs

## How to Run

1. Clone or download the repository.
2. Open the terminal or command prompt.
3. Compile the program

```bash
g++ InfoCollector.cpp -o InfoCollector
